# GroceryManagementUI

A Java Swing Grocery Management System with modern UI styling.

## Features
- Add, delete, and manage products
- Add, delete, and manage customers
- Record sales and calculate total revenue
- Stylish GUI with gradient panels and modern tables

## How to Run
1. Make sure you have Java installed.
2. Open `GroceryManagementUI.java` in your favorite IDE (Eclipse, IntelliJ, NetBeans).
3. Run the program.
